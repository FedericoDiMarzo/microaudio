Detailed Description

template<typename T, size_t CHANNEL_NUM, size_t BUFFER_LEN>
class AudioBuffer< T, CHANNEL_NUM, BUFFER_LEN >

This template class define a multi channel buffer that can be used to store and process audio.

Template Parameters
Ttype stored in the buffer // TODO: restrict T for numeric types
CHANNEL_NUMnumber of channels of the buffer
BUFFER_LENlength of each channel of the buffer

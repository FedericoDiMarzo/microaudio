Detailed Description

template<size_t CHANNEL_NUM>
class AudioModule< CHANNEL_NUM >

An AudioModule is an abstract template class that can be subclassed to implement a module that writes and processes an AudioBuffer.

Template Parameters
CHANNEL_NUMspecifies if the AudioModule is mono, stereo or multichannel

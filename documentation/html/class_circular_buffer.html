Detailed Description

template<typename T, size_t BufferSize, typename OverflowPolicy = CircularBufferType::Overwrite>
class CircularBuffer< T, BufferSize, OverflowPolicy >

This class implements a circular buffer with different overflow policies

Template Parameters
TType to be used in the collection
BufferSizeMax Buffer size of the circular buffer
OverflowPolicyDefaults to overwrite behaviour
